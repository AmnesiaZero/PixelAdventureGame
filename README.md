### Pixel adventure game
Пискельная игра в жанре adventure,сделанная на Java
### Реализованные фичи:

- [x] Базовые механики передвижения
- [x] Карта и окружение
- [x] UI
- [x] Музыка и звуковые эффекты
- [x] Система смерти и возрождений
- [x] Боевая система
- [x] Враги
- [x] NPC
- [x] Диалоги
- [x] Инвентарь
